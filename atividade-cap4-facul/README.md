# Fintech Dashboard

Projeto desenvolvido para a disciplina de Banco de Dados / Modelagem de Sistemas.

## Objetivo

Criar uma interface responsiva para uma Fintech utilizando:

- HTML5
- CSS3
- Tailwind CSS

## Funcionalidades

- Visualização de receitas
- Visualização de gastos
- Saldo disponível
- Metas financeiras
- Histórico de transações

## Responsividade

O sistema se adapta automaticamente para dispositivos móveis através do Tailwind CSS.
